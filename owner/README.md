# Owner 控制台

## 📁 文件说明

此文件夹包含项目 Owner (所有者) 专用的管理控制台文件。

### 文件列表

```
owner/
└── admin.html          - Owner 控制台主页面
```

## 🔒 安全说明

**⚠️ 重要**: 此文件夹包含管理员功能,仅供项目 Owner 使用!

### 访问限制

- ✅ **仅限 Owner 地址访问**
- ✅ 所有操作需要钱包签名验证
- ✅ 合约层面有 onlyOwner 权限检查

### 功能概览

#### 1. 矿池管理
- 💰 NFT 挖矿池余额查看和充值
- ⛏️ T-Engine 池余额查看和充值

#### 2. 紧急控制
- 🪙 **DPToken 控制**
  - 暂停/恢复代币转账
  - 查看供应量信息
  
- 🎫 **DebearPass 控制**
  - 暂停/恢复 Pass NFT 铸造
  - 查看各类 Pass 数量

#### 3. Pass 销售管理
- 设置 Pass 价格 (Standard/Premium/Exclusive)
- 开启/关闭销售
- 查看销售数据

#### 4. NFT 挖矿配置
- 设置基础日产出
- 暂停/恢复挖矿领取
- 配置 Claim 税费
- 设置税费接收地址

#### 5. T-Engine 配置
- 设置份额倍数
- 配置每日释放率
- 调整释放周期时长
- 设置退出税费
- 配置邀请奖励

#### 6. 数据统计
- 查看所有合约状态
- 实时数据刷新
- 操作日志记录

## 🔐 权限验证流程

1. 连接钱包 (OKX Wallet / MetaMask)
2. 验证网络 (Berachain 主网 80094)
3. 检查地址是否为 Owner
4. 显示管理界面

## 📊 合约地址

控制台会自动读取以下合约:

- **DPToken**: 代币合约
- **DebearPass**: Pass NFT 合约
- **PassSale**: Pass 销售合约
- **NFTMining**: NFT 挖矿合约
- **TEngine**: T-Engine 合约

## ⚡ 快速操作指南

### 充值矿池
```
1. 连接钱包
2. 输入充值金额
3. 点击"转入"按钮
4. 确认交易
```

### 设置价格
```
1. 输入 Standard/Premium/Exclusive 价格
2. 点击"设置价格"
3. 等待交易确认
```

### 紧急暂停
```
1. 点击对应的"暂停"按钮
2. 确认交易
3. 验证状态变更
```
